Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

.
Tag Name Purpose
p Paragraph Format text as a stand alone paragraph.
h1–h6 Headings 1 to 6 Format text as header with increasing font size and emphasis the lower the number.
table Table Format text into columns, in conjunction with the following table elements.
tr Table row Delineate the enclosed block as one row of the table.
th Table header Format the enclosed text as the header for a table column (not strictly a block element).
td Table data Format the enclosed text as a cell in the table (not strictly a block element).
header Header Labels a block section of the document, no direct formatting effect.
div Division Labels a block section of the document, no direct formatting effect.
ol Ordered list Format the enclosed text as an ordered list.
ul Unordered list Format the enclosed text as an unordered list
li List item Identify the enclosed text as a separate list item for either ol or ul
div Division This tag is used to apply uniform formatting to a block of text.
blockquote Block quote This tag is used to inform the browser that a block of text is a quotation from another source.

Inline Elements

Tag Name Purpose Example Result
em Emphasis Format the enclosed text to appear emphasised, typically with italic font. <em>Emphasised text</em> Emphasised text
strong Strong Format the enclosed text to appear strong, typically with boldface font. <strong>Strong text</strong> Strong text
a Anchor Format the enclosed text as a hyperlink. <a [options]>Hyperlink</a> Hyperlink
img Image Insert an image in line. <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code Code Format the enclosed text with fixed width font, like computer code. <code>Code listing</code> Code listing
span Span Span contains an inline section of text, for uniform application of formatting. <span>In line text</span> In line text
br Break Insert a line break in the text. Be not afraid of greatness.<br>Some are born great, Be not afraid of greatness.

Document Elements

Tag Name Purpose
html HTML This tag contains all of the document HTML code.
head Head This tag contains the document metadata, not for display.
body Body This tag contains the body of the document for display.
title Title This tag specifies the tab text for the browser.
meta Metadata This tag embeds useful data in the document, not for display.
